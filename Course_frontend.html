Usage Notes

  • Use the search bar to find by ID, course name, or instructor.
  • Inactive courses are hidden by default; enable Show inactive to see soft-deleted rows.
  • Click Edit to load a course into the form and then Save Changes.
  • Delete calls DELETE /delete/{id} and marks the course as inactive.
  • Dark/Light mode is remembered using localStorage.
